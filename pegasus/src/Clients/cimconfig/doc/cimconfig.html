cimconfig

Name

cimconfig - get, set, unset or list CIMOM configuration properties.

Synopsis

cimconfig -g name [ -c ][ -d ][ -p ]

cimconfig -s name=value [ -c ][ -p ]

cimconfig -u name [ -c ][ -p ]

cimconfig -l [ -c | -p ]

cimconfig -h

cimconfig --help

cimconfig --version

Description

The cimconfig command provides a command line interface to manage CIMOM configuration properties.

The first form of cimconfig allows to get current, planned and / or default value of the specified configuration property. 

The second form allows to set the current value and / or planned value of the specified configuration property to the specified value. 

The third form allows unsetting the current and / or planned values of  the specified configuration property to its default value.

The last form of this command allows for all the configuration properties to be listed. Specifying the -c or -p options, will provide a listing of all the current or planned configuration property names and values.

Specifying no options with the cimconfig command will show the usage message.

Options

The cimconfig command recognizes the following options:

-g                     Gets the current value of the specified configuration property.

-g -c              Gets the current value of the specified configuration property.

-g -p              Gets the planned value of the specified configuration property.

-g -d              Gets the default value of the specified configuration property. Returns an error when the CIMOM is not running.

-s                     Indicates that a configuration property is to be added or updated by setting its current value to the specified value. Returns an error when the CIMOM is not running or the specified property is not dynamically updatable.

-s -c              Indicates that a configuration property is to be added or updated by setting its current value to the specified value. Returns an error when the CIMOM is not running or the specified property is not dynamically updatable.

-s -p              Indicates that a configuration property is to be added or updated by setting its planned value to the specified value.

-u                     Indicates that the current value of the specified configuration property is to be reset to default. Returns an error when the CIMOM is not running or the specified property is not dynamically updatable.

-u -c              Indicates that the current value of the specified configuration property is to be reset to default. Returns an error when the CIMOM is not running or the specified property is not dynamically updatable.

-u -p              Indicates that the planned value of the specified configuration property is to be reset to default.

-l                     Displays the name of all the configuration properties.

-l -c              Displays the name and value pair of all the current configuration properties.

-l -p              Displays the name and value pair of all the planned configuration properties.

-h, --help          Displays the help message

--version           Displays CIM Server version number

Return Values

cimconfig returns one of the following values:

0         Successful completion.

1         Error

When an error occurs, an explanatory error message is written to stderr and an error value 1 is returned.

Examples

The following commands get the current value for the configuration property "port".

    cimconfig -g port

    cimconfig -g port -c

The following command gets the planned value for the configuration property "traceLevel".

    cimconfig -g traceLevel -p

The following commands set the current value of the property "traceLevel", to the new value "2".

    cimconfig -s traceLevel=2

    cimconfig -s traceLevel=2 -c

The following command sets the planned value of the property "traceLevel", to the new value "3".

    cimconfig -s traceLevel=3 -p

The following commands reset the current value of the property "traceLevel" to the default value.

    cimconfig -u traceLevel

    cimconfig -u traceLevel -c

The following command resets the planned value of the property "traceLevel" to the default value.

    cimconfig -u traceLevel -p

The following command displays the list of all the current configuration properties.

    cimconfig -l

The following command displays the list of all the current configuration properties and values.

    cimconfig -l -c

The following command displays the list of all the planned configuration properties and values.

    cimconfig -l -p

Limitations

This version of the cimconfig command does not return classified error codes. When an error occurs, it writes explanatory error message to stderr and returns an error value 1.
