The mof snippits herein test the following but only at the level of parsing.
These tests do not test the resulting information in the repository.

1. ValidMof1.mof scalar properties, Units
2. ValidMof2.mof Class with scalar properties with default values
3. ValidMof3.mof Comments arround and in mof
4. ValidMof4.mof Class with multiple instances
4. ValidMof4a.mof Instances of a Class
5. ValidMof4b.mof character escapes in string constants
6. ValidMof5.mof test of instances creation
7. ValidMof6.mof Class with arrays and instances with values.
8. ValidMof7.mof test association types
9. ValidMof8.mof Qualifiers for MinValue, MaxValue
